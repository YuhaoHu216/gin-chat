# 配置基本的开发环境
项目文件里创建main.go 终端输入命令
```shell
go mod init ginchat
go mod tidy
```
导入包
```shell
go get gorm.io/gorm
go get gorm.io/driver/mysql
```
网络相关
```shell
go get -u github.com/gin-gonic/gin
```
配置相关
```shell
go get github.com/spf13/viper
```
swagger
采用该swagger
https://github.com/swaggo/swag/blob/v2.0.0-rc5/README_zh-CN.md
```shell
go get -u github.com/swaggo/swag/v2/cmd/swag@latest
```
如果没设置环境变量的话init命令如下
```shell
D:\Code\go\bin\swag.exe init
```
